# 简介
23种设计模式，Java代码实现以及每种模式的总结笔记，持续更新中...

# 目录
```text
src
    ├─code       - 代码
    │  ├─adapter   - 适配器模式
    │  ├─bridge    - 桥接模式
    │  ├─builder   - 建造者模式
    │  ├─command   - 命令模式
    │  ├─composite - 组合模式
    │  ├─decorator - 装饰者模式
    │  ├─facade    - 外观模式
    │  ├─factory   - 工厂模式（包含抽象工厂）
    │  ├─flyweight - 享元模式
    │  ├─proxy     - 代理模式
    │  └─singleton - 单例模式
    │  └─template  - 模板模式
    └─note       - 各个模式的笔记（工厂、抽象工厂、原型、建造者模式都在「创建型模式总结.md」中）
```
